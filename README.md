## 📌 Overview
SecureHealth AI is a privacy-preserving medical image classification system that uses deep learning and encryption techniques.

- Detects multiple diseases from medical images (e.g., pneumonia, tuberculosis, COVID-19)
- Preserves patient privacy using Fully Homomorphic Encryption (FHE)
- Performs predictions directly on encrypted data
- Combines AI and cryptography for secure healthcare solutions

---

## 🌍 Features
- 🔐 Privacy-preserving inference using encryption  
- 🧠 Deep learning-based disease prediction  
- 🖼️ Supports medical image inputs (X-rays, scans)  
- 📂 Upload images for analysis  
- 📊 Displays prediction results with accuracy metrics  
- ⚡ REST API for real-time predictions
